Space Chase

Space Chase is an open source, cross platform, mobile multiplayer sample game created by us at Nextpeer. We created this game to demonstrate what the Nextpeer Multiplayer SDK is capable of, and how to do it. We try to provide as many tools we can for our developers so they can create the best games they can. Developers are now able see all the main functionalities of our SDK and review the code to really see how it is done.

Features:

  • Real Time Multiplayer - Users can experience real time tournaments against each.
  • Cross Platform - Players can interact across Android and iOS devices.
  • Seeing Opponents - Allowing users to see one another on screen provides for a much more exciting experience.
  • Power Ups - Players can attack and defend against one another, adding a whole new environment to the game.
  • Multiple Levels (And Random Seed) - Having a few different levels keeps players interested and the use of the random seed ensures all players are on the same level, seeing the same things.
  • Avatar Selection - Giving users different characters to select from lets them see know who they are and adds variation in the game.
