Space Chase

Open Source. Cross Platform. Mobile Multiplayer Game

Description:

Space Chase is an open source, cross platform, mobile multiplayer sample game created by us at Nextpeer. We created this game to demonstrate what the Nextpeer Multiplayer SDK is capable of, and how to do it. We try to provide as many tools we can for our developers so they can create the best games they can. Developers are now able see all the main functionalities of our SDK and review the code to really see how it is done.

Features:

  • Real Time Multiplayer
  • Cross Platform
  • Seeing Other Players
  • Power Ups
  • Multiple Levels (And Random Seed)
  • Avatar Selection
